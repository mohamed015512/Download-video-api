
# 🎥 Cup Video Downloader API

API قوي لتحميل الفيديوهات من YouTube وآلاف المواقع الأخرى.

## 🚀 التثبيت والتشغيل

### محلياً (Local)

```bash
# 1. استنساخ المشروع
git clone https://github.com/your-repo/video-downloader.git
cd video-downloader

# 2. إنشاء بيئة افتراضية
python -m venv venv
source venv/bin/activate  # على Windows: venv\Scripts\activate

# 3. تثبيت المكتبات
pip install -r requirements.txt

# 4. تشغيل السيرفر
python app.py
